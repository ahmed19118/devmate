مقدمة في JavaScript

JavaScript هي لغة البرمجة اللي بتضيف التفاعل للمواقع. لما تدوس على زر، أو يظهر إشعار، أو يتغير شيء في الصفحة بدون ما تعيد تحميلها، ده شغل جافاسكريبت.

مع HTML و CSS، بتشكل JavaScript الركن التالت لأي صفحة ويب احترافية.

مثال بسيط:

<script>
  alert("أهلاً بك في موقعي!");
</script>

الكود ده بيعرض رسالة ترحيبية أول ما الزائر يدخل الموقع.
