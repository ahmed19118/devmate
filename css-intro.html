مقدمة في CSS

CSS هي اختصار لـ Cascading Style Sheets وتعني "أوراق الأنماط المتتالية". تُستخدم لتنسيق صفحات الويب من حيث الألوان، الخطوط، ترتيب العناصر، وغيرها.

تُكتب CSS جنب HTML علشان تدي شكل جميل ومهني للموقع. بدل ما الصفحة تكون مجرد نصوص وصور، CSS بيخليها أنيقة وجذابة.

مثال بسيط:

<style>
  body {
    background-color: lightblue;
  }
</style>

السطر ده هيغير لون خلفية الصفحة إلى أزرق فاتح.
